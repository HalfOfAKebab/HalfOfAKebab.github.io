FAQ

Where can I download the archive?

Simply go to here, or join our discord server.

Have you heard of Luigi2600's Hidden Versions?

Yes, and a lot of them are either heavily modified or outright fake.
If you have downloaded his archive, it is highly recommended that you delete it and instead download our archive.

You know that the launcher has every version, right?

While the launcher does have some old versions, it isn't even close to every version. For example, at the time of writing this, there is only a single Infdev client in the launcher, yet we have 26.
Some of the launcher's versions are also slightly modified.

How do I get the archivist role in your Discord server?

The archivist role is used to thank people who have made significant contributions to the archive. Most of the time this is by finding versions, though some people have been granted the role for new discoveries that help us with our goal.

Do you have [insert version here]?

Check the index for a comprehensive list on which versions we do and do not have.

How do I know which servers are compatible with which clients?

Whether or not a client can connect to a server depends on the protocol version number (PVN) of both the client and the server.
If the PVN matches, they are compatible.
For example, clients a1.0.16 to a1.0.16_02 (inclusive) all have a PVN of 14.
Server versions a0.1.2_01 and a0.1.3 also have a PVN of 14, meaning that all those clients are compatible with both those servers.

To find the PVN of a client or server, take a look at the index.

Why do versions from Infdev and older freeze my entire PC?

This is a bug with Windows 10 version 1809. This is the only version this bug affects, so to fix it, either update to 1903 (or later), or downgrade to 1803 (or earlier).
To check what version of Windows 10 you're using, press Win+R, type winver, and hit enter. The dialogue box that opens will tell you the build of Windows 10 you're running.
There is no known way to fix this bug other than to install a build of Windows 10 that isn't affected.

I think I've found a new version; what do I do?

Firstly, upload it to Omnibot#4791 in DMs, with the message .identify. The bot will then hash the file and check the result against the index to identify the version. If it can't identify the version, then please send it to HalfOfAKebab#5139 in his DMs.

Here is a screenshot showing how to use .identify properly: Imgur.

Please do not upload jar files directly in the server, as explained in our server rules.

How can I be notified when a new version is found?

Every time a new version is found, it's announced in our discord's #changelog channel along with a ping to the Notification Squad role. To join the Notification Squad, type !role Notification Squad in #bots.
You can also use !role Voice Squad to join the Voice Squad role, which anyone can ping to let others know they want to voice chat.
Finally you can also be notified of upcoming events with the Event Squad role, to join use !role Event Squad.

How do I fix sounds, skins, and crashes?

As these versions get older, more things break as Mojang updates their own infrastructure and as computers get newer.
Not all of these things can be completely solved, however there are tools generously made by members of our community that iron out some of these problems.
Some solutions are listed below. Please note that you should only use one of these tools; using a combination is unsupported and may result in unpredictable or undesired behaviour.
To use a proxy, paste them into the JVM arguments field in your launcher profile.

BetaCraft
BetaCraft has a proxy and a launcher. The proxy only fixes skins, sounds, and crashes, whereas the launcher has more features.
These features include multiplayer authentication, resolution fixes in Classic, Discord RPC support, Classic saving, and more: Github.
BetaCraft launcher: Github
BetaCraft proxy: -Dhttp.proxyHost=betacraft.pl
It is also recommended to use the following JVM argument with the BetaCraft proxy: -Djava.util.Arrays.useLegacyMergeSort=true

RetroWrapper
RetroWrapper fixes some bugs, fixes skins and sounds, adds online saving, and has a teleportation hack and isometric viewer.
Find the download and installation instructions here.

Why do my graphics freak out on old versions?

Beta 1.7.3 and older have graphical bugs when using most AMD graphics cards, which is most noticeable when looking at clouds with Fancy graphics.

The BetaCraft launcher has a fix automatically applied without needing to modify your client jar file.
You can download the BetaCraft launcher here (direct download).

Alternatively, HalfOfAKebab has made a mod for a lot of versions that copies code from Beta 1.8 (which does not have these graphical issues). You can download it here.
